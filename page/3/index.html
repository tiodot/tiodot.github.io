Terminal 路径显示

查看提示符显示方式：

echo $PS1

修改$PS1，打开文件 ~/.bash_profile，增加一条规则，如果有则编辑即可：

export PS1=”\u \w$”

如果需要设置路径颜色，可以修改为：

export PS1=”[\e[33;1m\u@\h: \e[31m\W\e[0m]$“

保存修改并退出，应用更新：

source ~/.bash_profile
